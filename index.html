📘 PlanIA Edu

PlanIA Edu es una herramienta basada en inteligencia artificial diseñada para asistir a docentes en la planificación pedagógica de manera eficiente.

🚀 Características

  • Generación automática de planificaciones pedagógicas.
  • Exportación en formato PDF.
  • Historial organizado en JSON.
  • Compatible con Streamlit Cloud.

🔐 Configuración

Para ejecutar la aplicación, se requiere una clave de OpenAI. Consulta el archivo CONFIGURACION.md en este repositorio para configurarla correctamente.

📤 Despliegue

Sube el proyecto a tu cuenta de GitHub y conéctalo a Streamlit Cloud. El archivo principal es planiaedu.py.
